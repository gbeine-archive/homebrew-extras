# Overview #

[Homebrew](http://brew.sh)-formulae that did not make it into the main repository.

*   **oracle-client**
    - Oracle command line client for Mac OS X
*   **sane-frontends**
    - X11 frontends for SANE

## Install ##

*   `brew tap gbeine/homebrew-extras`
*   `brew install sane-frontends`
